### How to create a Lightning wallet?
Use wallets like: Wallet of Satoshi, Phoenix, Blue Wallet, Sparrow, etc. [Check the complete guide here](https://blog.areabitcoin.com.br/guia-de-carteiras-lightning/).

### What is Lightning Address?
It's like an "email" for receiving Bitcoin instantly. Example: `yourname@walletofsatoshi.com`

### What is Silent Payment?
A reusable Bitcoin address with enhanced privacy, without exposing your transaction history.
